# Animal Sound Categories in ESC-50

The ESC-50 dataset contains 10 animal sound categories:

1. **Dog** - Dog barking
2. **Rooster** - Rooster crowing  
3. **Pig** - Pig sounds
4. **Cow** - Cow mooing
5. **Frog** - Frog croaking
6. **Cat** - Cat meowing
7. **Hen** - Hen clucking
8. **Insects** - Flying insects (bees, mosquitoes)
9. **Sheep** - Sheep bleating
10. **Crow** - Crow cawing

Each category contains 40 audio clips (5 seconds each), totaling **400 animal sound samples** from the 2000 clips in ESC-50.
